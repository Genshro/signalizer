🎯 Proje Amacı
Kullanıcıların kripto para borsalarından alınan anlık fiyat verileri ile özelleştirilmiş indikatörlerin analiz ettiği alım-satım bölgelerini görebileceği bir Android & iOS mobil uygulama geliştirmek.

Alım bölgeleri, hedef fiyatlar, stop-loss gibi dinamik sinyaller üretilecek ve kullanıcıya gösterilecek.

🚀 Öne Çıkan Özellikler
✅ Gerçek zamanlı coin fiyat takibi
✅ Özel indikatör hesapları (örn: RIBQA, GCR fractal, Ricci Flow bazlı sinyal)
✅ Alım bölgesi, hedef1-2-3, stop-loss önerileri
✅ Bildirim (push notification) ile sinyal duyuruları
✅ Kullanıcı favori coinlerini seçebilecek
✅ Çoklu para birimi desteği (USD, EUR, TRY vs)
✅ Temiz UI/UX: dark & light mode

🏗️ Teknoloji Yığını (Tech Stack)
Katman	Önerilen Teknoloji
📱 Mobil app	Flutter (Dart) → Android & iOS tek codebase
🌐 API	Go (Gin veya Fiber) REST API
🔄 Real-time	Binance WebSocket + Redis pub/sub
🗃 DB	PostgreSQL (indikator parametreleri, user preferences)
⚡ Cache	Redis (tick data, sinyal cache)
🧠 AI/Calc	Go backend içinde custom indicator libraries
📈 Data source	Binance, KuCoin, CoinGecko API
🔔 Push notif.	Firebase Cloud Messaging (FCM)
🔒 Auth	JWT + Supabase (veya kendi auth db)
🖥 CI/CD	GitHub Actions + Docker build
📝 Monitor	Sentry (error log), Grafana (metrics)

🧩 Temel Ekranlar & İşlevler
Ekran / Modül	Özellikler
📊 Dashboard	BTC, ETH, SOL gibi favori coinlerin anlık fiyatı + son sinyal durumu
🔍 Coin Detail	Seçilen coin için alım bölgesi, hedef1-2-3, stop-loss
⚙ Settings	Para birimi seçimi, dil, bildirim tercihleri
🔔 Notifications	Yeni sinyal geldiğinde göster
💎 About / Licenses	Versiyon, gizlilik, kullanım koşulları

🔥 İş Akışı (High-Level Flow)
Mobil app açılır →

Go backend’den GET /coins çağrısı ile anlık fiyat & sinyal dataları gelir →

Kullanıcının favori coinleri filtrelenir →

🔐 Güvenlik & Performans
JWT ile mobil login → user bazlı favoriler, ayarlar.

Redis cache → Websocket data’yı anında analiz.

Rate limiting & anti-DDOS (Gin middleware veya Traefik/Nginx).
